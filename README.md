# Scala Functional Programming

A collection of functional programming implementations 
in Scala, covering algorithms, language theory, 
interpreters and game logic. Scored 94/100.

## Modules

**Collatz Conjecture (Core 1)**
Tail-recursive implementation of the 3n+1 sequence 
with maximum finder and power-of-two analysis.

**Document Similarity (Core 2)**
Cosine similarity engine using word frequency vectors 
and dot products to measure text overlap — applicable 
to plagiarism detection.

**Shunting Yard Algorithm (Core 3)**
Dijkstra's algorithm to convert infix expressions to 
postfix notation with a stack-based evaluator.

**Evil Wordle (Main 2)**
Adversarial Wordle engine that dynamically selects 
the hardest secret word using scoring, frequency 
analysis and ranking functions.

**Regular Expression Engine (Main 3)**
Brzozowski derivative-based regex matcher with 
algebraic simplification to prevent exponential 
blowup on complex patterns.

**Shogun Board Game (Main 4)**
Complete move generation, attack detection and legal 
move validation for a chess-variant board game with 
energy-based piece mechanics.

**Brainf*** Interpreter (Main 5)**
Full interpreter for the esoteric BF language with 
jump optimisation, memory management and a BF 
code generator.

## Stack
Scala 3 — purely functional, no mutable state
